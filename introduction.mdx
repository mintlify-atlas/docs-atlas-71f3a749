## What is GitaChat?

GitaChat is a semantic search platform that helps you find relevant wisdom from the Bhagavad Gita's 700+ verses. Ask a question about life, work, relationships, or any challenge you're facing, and receive guidance from ancient teachings paired with modern AI-generated contextual commentary.

## How It Works

GitaChat combines traditional spiritual wisdom with modern AI technology:

<Steps>
  <Step title="Ask Your Question">
    Enter any question or situation you're facing. GitaChat converts it into a vector embedding.
  </Step>
  <Step title="Semantic Search">
    The system searches across all 700+ verses in Pinecone vector database to find the most relevant matches.
  </Step>
  <Step title="AI Commentary">
    OpenAI generates contextual commentary that explains how the verse applies to your specific situation.
  </Step>
  <Step title="Explore & Save">
    Read traditional commentary alongside AI-generated insights. Bookmark verses, take notes, and explore related teachings.
  </Step>
</Steps>

## Open Source

GitaChat is open source and built with modern technologies:

- **Frontend**: Next.js, TypeScript, Tailwind CSS
- **Backend**: Python, FastAPI, Sentence Transformers
- **Database**: Pinecone (vector DB), Supabase (user data)
- **AI**: OpenAI GPT-4o-mini for contextual commentary

View the source code on [GitHub](https://github.com/aadit2805/gitachat.org) and contribute to the project.
